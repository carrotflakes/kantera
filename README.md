# kantera
Video composition and rendering kit for Rust, depends on FFmpeg and cairo.

![kantera-logo](out.jpg)

## Usage

For development:
``` sh
$ cargo build --release --bin && cargo run --release
```

## Installation

- FFmpeg
- cairo
